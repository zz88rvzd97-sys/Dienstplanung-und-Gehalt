    Cloud, Backup & Sicherheit

    Die App ist erst nach Login sichtbar. Nach erfolgreicher Anmeldung werden Änderungen in Supabase gespeichert und lokal nur als Zwischenspeicher gehalten.

    Wichtig: In Supabase muss Row Level Security aktiv sein, damit jeder angemeldete Nutzer ausschließlich seine eigenen Daten lesen und speichern kann.

    Die Krankenkassenliste enthält 93 gesetzliche Krankenkassen mit Zusatzbeiträgen laut GKV-Spitzenverband, Stand 25.04.2026. Die Prozentwerte bleiben editierbar; wenn eine echte Monatsabrechnung hinterlegt ist, kann die App diese echten Abzüge bevorzugen.

    Gefahrenzone

    Setzt lokale und Cloud-Daten auf den leeren Standard zurück. Vorher am besten ein Backup exportieren.
