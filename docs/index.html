    Hinweise

    Diese GitHub-Pages-Version läuft komplett statisch ohne Build-Schritt. Alle Daten bleiben lokal im Browser, bis du ein Backup exportierst.

    Seitenlogik enthalten: Umlauf, FRV, Tage, Lohn, Abrechnungen, Fahrten & Steuer, Jahresauswertung.

    Fixe Grundvergütung 3.022 €, Zuschläge im Folgemonat, Feiertag 100 % + 35 %, Urlaub/LFZ mit Faktor × 7,8, KVBbg 0,55 % + 2,40 % auf ZV-Brutto.

    Neu: direkte Abgaben-Schätzung mit Steuerklasse und Krankenkassen-Auswahl. Die Krankenkassenliste enthält 93 gesetzliche Krankenkassen mit Zusatzbeiträgen laut GKV-Spitzenverband, Stand 25.04.2026. Die Prozentwerte bleiben editierbar; wenn eine echte Monatsabrechnung hinterlegt ist, kann die App diese echten Abzüge bevorzugen.

    Für GitHub Pages reicht diese docs-Struktur. Einfach im Repo den vorhandenen docs-Ordner ersetzen und pushen.
